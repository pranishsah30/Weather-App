# Weather-App
A simple and user-friendly weather application that displays real-time weather information.

## Features
- Fetches weather details for any city
- Displays temperature in Celsius
- Shows weather condition
- Displays humidity percentage
- Shows rainfall data
- Displays wind speed
- Shows cloud / overcast percentage

## Technologies Used
- HTML
- CSS
- Weather API
